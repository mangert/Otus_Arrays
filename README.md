# Структуры данных

Учебный проект по реализации базовых структур данных на C++.

## Структура проекта

### Основные структуры данных
- **Массивы**: 
  - `Array` - базовый класс
  - `SingleArray` - простой массив
  - `VectorArray` - векторный массив  
  - `FactorArray` - факторный массив
  - `MatrixArray` - матричный массив

- **Очереди**:
  - `PriorityQueue` - очередь с приоритетами

- **Списки**:
  - `List` - базовый список

### Тестирование
- `TestArrays` - тестирование массивов и списков
- `QueueTester` - тестирование очереди с приоритетами
- `main.cpp` - точка входа с демонстрацией работы

## Особенности
- Все реализации шаблонные (template)
- Проект собран на CMake
- Включает тестирование всех структур
- Многопоточное тестирование для очереди
